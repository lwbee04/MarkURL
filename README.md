📖 About MarkURL
MarkURL is a lightweight, self-hosted URL shortener built with PHP and MySQL. Perfect for individuals, businesses, and developers who want full control over their shortened links with built-in analytics tracking.<br>
✨ Why Choose MarkURL?<br>
🏠 Self-Hosted - Full control over your data and links<br>
📊 Analytics - Track clicks and engagement in real-time<br>
🔒 Secure - Admin-only access with password hashing<br>
⚡ Fast - Lightweight code, minimal dependencies<br>
🎨 Modern UI - Clean, responsive design out of the box<br>
🔄 Auto Updates - Remote file deployment system<br>
🎯 Features<br>
<br>
<br>
🔐 Security Features<br>
✅ Admin Authentication - Secure login with bcrypt password hashing<br>
✅ SQL Injection Protection - All queries use prepared statements<br>
✅ XSS Protection - Output escaping on all user inputs<br>
✅ Session Management - Secure session handling<br>
✅ File Protection - Sensitive files blocked via .htaccess<br>
<br>
<br>
📊 Analytics & Tracking<br>
✅ Click Counter - Track total clicks per URL<br>
✅ Last Clicked - See when each link was last accessed<br>
✅ Real-time Stats - View analytics in admin dashboard<br>
✅ Popular Links - Identify your most clicked URLs<br>
