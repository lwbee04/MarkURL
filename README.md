📖 About MarkURL
MarkURL is a lightweight, self-hosted URL shortener built with PHP and MySQL. Perfect for individuals, businesses, and developers who want full control over their shortened links with built-in analytics tracking.
✨ Why Choose MarkURL?
🏠 Self-Hosted - Full control over your data and links
📊 Analytics - Track clicks and engagement in real-time
🔒 Secure - Admin-only access with password hashing
⚡ Fast - Lightweight code, minimal dependencies
🎨 Modern UI - Clean, responsive design out of the box
🔄 Auto Updates - Remote file deployment system
🎯 Features


🔐 Security Features
✅ Admin Authentication - Secure login with bcrypt password hashing
✅ SQL Injection Protection - All queries use prepared statements
✅ XSS Protection - Output escaping on all user inputs
✅ Session Management - Secure session handling
✅ File Protection - Sensitive files blocked via .htaccess


📊 Analytics & Tracking
✅ Click Counter - Track total clicks per URL
✅ Last Clicked - See when each link was last accessed
✅ Real-time Stats - View analytics in admin dashboard
✅ Popular Links - Identify your most clicked URLs


🔗 URL Management
✅ Custom Short Codes - Create memorable short URLs
✅ Auto-Generate - Random 6-character codes
✅ Edit URLs - Change destination without new link
✅ Delete URLs - Remove unwanted links
✅ Pagination - 5 URLs per page for easy navigation
🎨 User Interface
✅ Responsive Design - Works on all devices
✅ Modern UI - Clean gradient design
✅ Toast Notifications - Real-time feedback
✅ One-Click Copy - Copy short URLs instantly
✅ Dark Mode Ready - Easy to customize theme
