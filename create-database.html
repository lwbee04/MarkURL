Tutorial Create Database di cPanel

Panduan lengkap step-by-step membuat database MySQL untuk MarkURL URL Shortener

Home / Tutorial / Create Database

Tentang Tutorial Ini

Tutorial ini akan memandu Anda membuat database MySQL di cPanel yang diperlukan untuk menginstall MarkURL URL Shortener. Ikuti setiap langkah dengan seksama.

Yang Anda Butuhkan:

  • Akses ke cPanel hosting Anda
  • Username dan password cPanel
  • Waktu sekitar 5-10 menit
  • Tidak perlu pengalaman coding
1

Login ke cPanel

Langkah pertama adalah login ke cPanel hosting Anda. Biasanya cPanel dapat diakses melalui:

https://domain-anda.com/cpanel
atau
https://cpanel.domain-anda.com
atau
https://ip-server:2083

Masukkan username dan password cPanel Anda, lalu klik Login.

Tips:

  • Jika lupa password, gunakan fitur "Forgot Password" di halaman login
  • Simpan kredensial cPanel Anda di tempat yang aman
  • Gunakan koneksi internet yang stabil

3

Buat User Database

Setelah database dibuat, sekarang buat user yang akan mengakses database tersebut.

3.1. Create New User

Scroll ke bagian MySQL Users > Add New User:

c
cPanel
Home / Databases / MySQL Databases
Add New User
username_
Strength: Strong

Penting!

Simpan username dan password database di tempat yang aman. Anda akan membutuhkannya saat instalasi MarkURL!

Tips Keamanan Password:

  • Gunakan minimal 12 karakter
  • Kombinasi huruf besar, huruf kecil, angka, dan simbol
  • Jangan gunakan password yang mudah ditebak
  • Gunakan password generator jika perlu
4

Tambahkan User ke Database

Langkah terakhir adalah memberikan akses user ke database yang sudah dibuat.

4.1. Add User To Database

Scroll ke bagian Add User To Database:

c

Home / Databases / MySQL Databases
Add User To Database

4.2. Set Privileges

Pada halaman berikutnya, berikan semua privileges:

Manage User: username_roll_user on Database: username_roll_pemindah

Selesai!

Database dan user sudah siap digunakan. Catat informasi berikut untuk instalasi MarkURL:

Ringkasan Informasi Database

Simpan informasi ini untuk digunakan saat instalasi MarkURL:

Informasi Keterangan Contoh
Database Host Server database localhost
Database Name Nama database yang dibuat username_roll_pemindah
Database Username User database yang dibuat username_roll_user
Database Password Password user database ••••••••••••

Catatan Penting:

Database Host biasanya adalah localhost. Namun beberapa hosting menggunakan server database terpisah. Jika localhost tidak berhasil, hubungi support hosting Anda untuk informasi yang tepat.

Langkah Selanjutnya

Setelah database berhasil dibuat, lanjutkan dengan instalasi MarkURL:

  1. Upload file install.php ke hosting Anda
  2. Akses https://domain-anda.com/install.php
  3. Masukkan informasi database yang sudah dibuat
  4. Klik Install dan selesai!
